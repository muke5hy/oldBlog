Creating logo online

How to create logo online free

  1. Create logo on below site https://www.graphicsprings.com/start-your-logo

  2. Get the SVG code

  3. Edit the svg code at below tool http://svg-edit.googlecode.com/svn/branches/stable/editor/svg-editor.html

  4. Now you can export as PNG also you can open the image on GIMP.
